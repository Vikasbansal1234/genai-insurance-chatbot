# Test Setup

## Installation

Install test dependencies:

```bash
npm install
```

## Running Tests

```bash
# Run all tests
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with coverage
npm run test:cov
```

## Test Structure

Tests follow the same folder structure as the source code:

```
test/
  domain/
    agent/
      agent.controller.spec.ts
```
